# Frontend - React Vite

Frontend application menggunakan React JS dengan Vite sebagai build tool.

## Technology Stack

- ReactJS (Create React App atau Vite)
- React Router
- Axios/Fetch API
- CSS murni (wajib untuk CPMK0501) - boleh tambah Tailwind/Bootstrap

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

Server akan berjalan di `http://localhost:3000`

## Build

```bash
npm run build
```

## Project Structure

```
src/
├── components/     # Reusable components
├── pages/          # Page components
├── services/       # API calls
├── App.jsx         # Main App component
├── App.css         # App styles
├── index.js        # Entry point
└── index.css       # Global styles
```
